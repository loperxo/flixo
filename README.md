# Flixo

Flixo is a web interface for a dataset of Netflix TV shows and movies that has the ability to sort and filter by the type, title, year, or director fields.

View Demo [Here](https://flixo.netlify.app/) - (flixo.netlify.app)

[![Netlify Status](https://api.netlify.com/api/v1/badges/61676889-e2f1-4a81-b290-78f4c1cd19cb/deploy-status)](https://app.netlify.com/sites/flixo/deploys)

## Dev

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# run tests
$ yarn test

# build for production and deploy
$ yarn run generate
```
